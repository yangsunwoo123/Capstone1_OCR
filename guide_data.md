## 점검 사항

### 1. 라벨 포맷 확인 (확인 필요)
- [ ] 라벨 파일 확장자 (`.json`, `.txt`, `.csv` 등)
- [ ] 단일 파일인지, 이미지당 1개 파일인지
- [ ] 라벨 필드 구조 (텍스트, bbox, 작성자 정보 등)
- [ ] 한글 인코딩 (UTF-8 여부)

### 2. 이미지-라벨 매칭
- [ ] 파일명 기준 매칭 규칙
- [ ] 매칭 실패 케이스 (이미지만 있거나 라벨만 있는 경우)

### 3. 이미지 속성
- [ ] 이미지 포맷 (`.png`, `.jpg` 등)
- [ ] 해상도 분포
- [ ] 채널 (RGB / Grayscale)
- [ ] 단어/줄/문단 중 어느 단위인지

### 4. 데이터 분포
- [ ] 전체 샘플 수
- [ ] 일반 손글씨 / 노인 손글씨 구분이 라벨에 포함되어 있는지
- [ ] 글자 길이 분포
- [ ] 빈 라벨 / 이상 샘플 개수

### 5. 데이터셋 분할 (결정 필요)
- [ ] train / validation / test 비율
- [ ] 분할 기준 (작성자 기준 / 랜덤)
- [ ] 노인 손글씨를 어느 분할에 포함할지 (모델링 계획과 연동)

### 6. 전처리
- [ ] ko-trocr 입력 크기 (모델 spec 확인)
- [ ] 정규화 방식
- [ ] 배경 제거 / 이진화 여부
