## API 엔드포인트 (설계 필요)
PROJECT.md의 사용 과정 4단계를 기반으로 한 최소 엔드포인트:

- [ ] `GET /forms` — 양식 목록 조회 (DB에서 동적으로 로드)
- [ ] `GET /forms/{id}` — 선택한 양식 상세 로드
- [ ] `POST /upload` — 스캔 이미지 업로드 (다중)
- [ ] `POST /recognize` — 업로드된 이미지 인식 요청
  - 응답: 텍스트, bbox, confidence, 저신뢰도 후보 리스트
- [ ] `POST /save` — 작성 완료된 양식을 암호화하여 저장
