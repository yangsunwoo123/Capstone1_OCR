- [ ] 스타일 방식 (Tailwind / CSS / UI 라이브러리)

## 레이아웃 (초기 화면)
- [ ] **좌측**: 파일 업로드 영역
- [ ] **우측**: 양식 선택 및 검색 (체크박스 포함)
- [ ] **중앙 하단**: 인식 버튼

---

## 파일 업로드
- [ ] 드래그앤드랍 지원
- [ ] 파일 선택 버튼 지원
- [ ] **다중 업로드** 필수
- [ ] 업로드 후 인식 버튼 상태 전환: **회색 → 파란색**

---

## 인식 진행 표시
- [ ] 진행 progress bar
- [ ] 진행 % 숫자 표시

---

## 인식 완료 후 화면

### 좌측
- [ ] 업로드한 이미지 표시
- [ ] 인식된 손글씨 영역에 **라운드박스 or 밑줄**
  - 일반 인식: **파란색**
  - 저신뢰도(인식 score 낮음): **빨간색**
- [ ] 저신뢰도 영역 바로 아래에 **후보 텍스트 제시**
  - iPhone 텍스트 제안과 유사한 스타일

### 우측
- [ ] 선택한 양식에 인식 결과가 채워진 **프리뷰**
- [ ] **직접 타이핑으로 수정 가능**한 상태

### 다중 이미지 네비게이션
- [ ] 이미지 하단에 **좌우 화살표**
- [ ] **번호 표시**로 다른 이미지 이동

---

## 저장
- [ ] 우측 양식 기준으로 전산(임시 DB / Google Drive)에 저장
- [ ] 저장 버튼 위치: 화면 하단

## 일반 점검
- [ ] 라운드박스 좌표 렌더링 방식 (SVG overlay / Canvas 등) — 결정 필요
- [ ] 저신뢰도 후보 UI가 다른 요소와 겹치지 않는지
- [ ] 다중 이미지 전환 시 우측 편집 내용 유지 여부 — **결정 필요**
- [ ] 본인확인 등 공무원 작업 편의를 위한 키보드 단축키 — PROJECT.md 미명시, 자의 추가 금지

## 주의
