# CLAUDE.md

이 파일은 프로젝트 작업 시 Claude가 참고할 수 있는 참고서들의 목차입니다.
각 참고서는 프로젝트의 특정 측면에 대한 점검 사항과 고려 사항을 담고 있습니다.

---

## 참고서 목록

### guide_data.md — 데이터 점검 참고서
- 경로
- 라벨(annotation) 포맷 확인
- 이미지-라벨 매칭
- 이미지 속성
- 데이터 분포
- 데이터셋 분할
- 전처리

### guide_model.md — 모델링 참고서
- 사용 모델
- Zero-shot 평가 (MVP 1단계)
- Fine-tuning — 일반 손글씨 (MVP 2단계)
- 노인 손글씨 추가 Fine-tuning (MVP 3단계)
- Development 단계
- 평가 지표
- Inference 시 추출해야 하는 정보

### guide_backend.md — 백엔드 개발 참고서
- 결정 필요 항목
- API 엔드포인트 설계
- 양식(Form) 관리
- 모델 연동
- 저장 (보안 연동)
- 일반 점검

### guide_frontend.md — 프론트엔드 개발 참고서
- 결정 필요 항목
- 레이아웃 (초기 화면)
- 파일 업로드
- 인식 진행 표시
- 인식 완료 후 화면 (좌측 / 우측 / 다중 이미지)
- 저장
- 일반 점검

### guide_security.md — 보안/개인정보 참고서
- 전제
- 저장 파일 보호
- 개인정보 마스킹
- 접근 제어
- 주의

### guide_integration.md — 통합 참고서
- 데이터 흐름 (양식 선택 → 업로드 → 인식 → 저장)
- 점검 사항

---

## 문서 사용 규칙
- 각 참고서는 점검 사항을 담은 체크리스트 역할을 한다.
- PROJECT.md에 명시되지 않은 항목은 "결정 필요" 또는 "확인 필요"로 표시되어 있다.
- 사용자가 명시하지 않은 기능/가정은 참고서에 포함하지 않는다.
- 작업 단계별 활용 방법은 `directions.md`를 참조한다.
